# User Authentication System (Flask + MySQL)

This is a full-stack web application built using Flask and MySQL.

## Features
- User registration
- User login
- MySQL database integration
- Simple authentication system

## Technologies Used
- Python (Flask)
- MySQL
- HTML

## How to Run
1. Install dependencies:
   pip install flask mysql-connector-python

2. Run the application:
   python app.py

3. Open browser:
   http://127.0.0.1:5000

## Project Purpose
This project demonstrates basic user authentication using Flask and MySQL.
